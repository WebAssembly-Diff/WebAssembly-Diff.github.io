Documentation for WasmDiff

    Introduction to WasmDiff

    WasmDiff is a discrepancy detector that is based on differential fuzzing to automate the detection of discrepancy exhibited by C/C++ code porting to Wasm.

    Discrepancy examples

    WasmDiff focuses on the functional inconsistencies emerging from the porting of C/C++ code. Here are the code examples of all types of discrepancies identified by WasmDiff:
    https://webassembly-diff.github.io/#Discrepancy-examples.html
